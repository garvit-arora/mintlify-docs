---
title: Chat completions
description: Send OpenAI-compatible chat requests through the DevQuest gateway.
---

## Endpoint

```http
POST /v1/chat/completions
```

## Request

```bash
curl http://localhost:8010/v1/chat/completions \
  -H "Authorization: Bearer dq_live_your_key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "devquest-gpt-56-sol",
    "messages": [
      { "role": "user", "content": "Explain this repo in three bullets." }
    ],
    "max_tokens": 600
  }'
```

## JavaScript

```ts
const response = await fetch("http://localhost:8010/v1/chat/completions", {
  method: "POST",
  headers: {
    Authorization: `Bearer ${process.env.DEVQUEST_API_KEY}`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    model: "devquest-gpt-56-sol",
    messages: [{ role: "user", content: "Write a short changelog." }],
  }),
});

const data = await response.json();
console.log(data.choices[0].message.content);
```

## Billing

Each successful request deducts at most 2 credits, regardless of token count.

Failed provider requests release the reserved credits.
